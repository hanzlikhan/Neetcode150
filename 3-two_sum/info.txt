You're given:
- An array `nums`
- A number `target`

Find **two indices** `i` and `j` such that:
- `nums[i] + nums[j] == target`
- `i â‰  j`
- Return the indices with **smaller index first**

---

### âœ… **Approaches to Solve**

#### âœ… 1. Brute Force (Nested Loops)

**Idea:**
- Try every pair of elements.
- Check if they add up to the target.

**Time:** `O(n^2)`  
**Space:** `O(1)`

```python
def two_sum_brute(nums, target):
    for i in range(len(nums)):
        for j in range(i + 1, len(nums)):
            if nums[i] + nums[j] == target:
                return [i, j]
```

---

#### âœ… 2. HashMap (Efficient)

**Idea:**
- Use a HashMap to store numbers we've seen so far.
- For every element `x`, check if `target - x` is already in the map.

**Time:** `O(n)`  
**Space:** `O(n)`

```python
def two_sum_hashmap(nums, target):
    seen = {}  # stores value: index
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
```

---

### âœ… Examples

```python
print(two_sum_hashmap([3, 4, 5, 6], 7))   # â [0, 1]
print(two_sum_hashmap([4, 5, 6], 10))     # â [0, 2]
print(two_sum_hashmap([5, 5], 10))        # â [0, 1]
```

public class TwoSum {
    public static int[] twoSumBrute(int[] nums, int target) {
        for (int i = 0; i < nums.length; i++) {
            for (int j = i + 1; j < nums.length; j++) {

}

#### ğŸ”¹ **Using HashMap (Java - Optimal)**
```java
import java.util.HashMap;

public class TwoSum {
    public static int[] twoSumHashMap(int[] nums, int target) {
        HashMap<Integer, Integer> map = new HashMap<>(); // value -> index

        for (int i = 0; i < nums.length; i++) {
            int complement = target - nums[i];

            if (map.containsKey(complement)) {
                return new int[]{map.get(complement), i};
            }

            map.put(nums[i], i);
        }

        return null;
    }
}
```
